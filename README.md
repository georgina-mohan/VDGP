# Visual Discomfort: General Population (VDGP)

This repository contains MATLAB code for the VDGP study, which collects visual discomfort ratings in response to gratings varying in spatial frequency, luminance contrast, chromatic separation and colour-pair. 

The experiment can be run with behavioural ratings only, and optionally include EEG and/or eye-tracking.

## Requirements:
- MATLAB R2021a or later
- PsychToolBox
- GStreamer
